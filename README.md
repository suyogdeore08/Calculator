# Calculator App

A fully functional calculator application built using **HTML**, **CSS**, and **Vanilla JavaScript**.

## Features

* Basic arithmetic operations:

  * Addition (+)
  * Subtraction (-)
  * Multiplication (*)
  * Division (/)
* Dynamic expression handling
* Real-time display updates
* Clear (C) functionality
* Continue calculations using previous results
* Responsive grid-based button layout
* Modern user interface with hover and click animations
* Clean and intuitive design

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## How It Works

1. User clicks number and operator buttons.
2. The expression is stored as a string.
3. The display updates dynamically as the expression changes.
4. Pressing "=" evaluates the expression and displays the result.
5. The result is stored and can be used in further calculations.
6. Pressing "C" clears the display and resets the calculator.
